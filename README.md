Aiti Guru

Как запустить

```bash
npm install
npm run dev
```

Дополнительно:

- `npm run build` — проверка TypeScript и сборка для продакшена
- `npm run preview` — посмотреть собранную версию локально
- `npm run lint` — прогнать ESLint

Адрес API по умолчанию задан в коде (`api/client.ts`). При необходимости бэкенд можно вынести в переменные `VITE_*` и читать через `import.meta.env`.

Что за стек и почему так

- **React** — база интерфейса: экраны, формы, таблица. В проекте актуальная ветка (React 19), чтобы соответствовать ТЗ (React 18+).
- **TypeScript (strict)** — явные типы для API и пропсов, меньше ошибок до запуска.
- **Vite** — быстрый dev и простая сборка SPA.
- **React Router** — страницы входа и товаров, защита маршрутов.
- **TanStack Query** — загрузка списка: кэш, повторы, индикатор загрузки, кнопка «обновить» без ручных `useEffect` на каждый запрос.
- **Zustand** — сессия, remember-me, сортировка с persist, выбор строк, toast; без лишнего Redux на объёме задачи.
- **CSS Modules** — стили рядом с компонентами, без глобальных коллизий классов.
- **ESLint** — базовые правила для TS/JSX и React.
